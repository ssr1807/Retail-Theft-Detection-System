# Retail Theft Detection System

B.Tech Training Project (04 May 2026 - 03 July 2026)

## Objective

Develop an intelligent retail theft detection system using computer vision and object tracking techniques to identify suspicious customer behaviour from surveillance footage.

## Technologies

- Python
- PyTorch
- YOLOv8
- DeepSORT
- OpenCV
- MOT17 Dataset

## Pipeline

Input Video
↓
YOLOv8 Detection
↓
DeepSORT Tracking
↓
Movement History
↓
Suspicious Behaviour Analysis
↓
Alert Generation
